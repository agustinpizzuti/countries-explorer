# Countries Explorer

**Countries Explorer** es una aplicación web desarrollada en **React** que permite explorar información de países de todo el mundo utilizando la API de **REST Countries**.

La aplicación permite visualizar países, buscarlos por nombre, filtrarlos por continente y navegar entre ellos mediante paginación.

---

## Funcionalidades

- **Listado de países** obtenido desde la REST Countries API.
- **Búsqueda por nombre** para encontrar países rápidamente.
- **Filtro por continente** para mostrar solo los países de una región específica.
- **Paginación** que permite navegar entre los países en grupos de 30.
- **Detalle de país**: al hacer clic en un país se muestra información más detallada.

---

## 🛠 Tecnologías utilizadas

- **React**
- **JavaScript**
- **REST Countries API**
- **CSS**

---

## Arquitectura

El proyecto sigue una estructura simple separando responsabilidades:

- **Componentes React** → Encargados de la interfaz de usuario.
- **Services** → Gestionan la obtención de datos desde la API.
- **Utils** → Contienen lógica reutilizable como la paginación.

Esto permite mantener el código más limpio, reutilizable y fácil de mantener.
